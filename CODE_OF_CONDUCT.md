# Code of Conduct

## Our Standard

We want this project to be a respectful, collaborative, and welcoming space.

Examples of positive behavior:

- Being respectful in discussions and reviews
- Giving constructive feedback
- Assuming good intent
- Helping others learn

Examples of unacceptable behavior:

- Harassment or abusive language
- Personal attacks
- Trolling or intentionally disruptive behavior
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and other contributions that do not align with this code of conduct.

## Scope

This code of conduct applies within project spaces and in public interactions representing the project.
