# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue first.

Instead, report it privately through GitHub Security Advisories or contact the maintainer directly through the repository owner account.

When reporting, please include:

- A short description of the issue
- Steps to reproduce
- Potential impact
- Any proof-of-concept or screenshots that help explain the problem

We will review reports as quickly as possible and coordinate a fix before public disclosure when appropriate.
